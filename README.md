## Описание
Проект на Spring Boot, демонстрирующий переход от классического MVC-приложения к архитектуре с REST-контроллерами и клиентской логикой на JavaScript.  
В проекте вся работа с данными (получение, добавление, редактирование, удаление) выполняется через REST API, а пользовательский интерфейс динамически обновляется без перезагрузки страницы.

CRUD-операции реализованы с помощью REST-контроллеров, которые возвращают данные в формате JSON.  
На стороне клиента используется Fetch API (или jQuery) для асинхронных HTTP-запросов, обновления таблиц, работы с модальными окнами и формами.  

Проект демонстрирует современный подход к веб-разработке, где backend и frontend взаимодействуют через REST, а сервер отвечает только за данные и бизнес-огику.
## Используемые технологии
- Java
- Spring Boot
- Spring REST Controllers
- Spring MVC
- Hibernate ORM
- EntityManager
- MySQL
- JavaScript (Fetch API / jQuery)
- JSON
- Maven
- Thymeleaf (как базовый шаблон)
- AJAX / SPA-like interaction
